Portfolio Section

This section contains the portfolio's items.

Portfolio Section have two parts:

  • Portfolio filters
  • Portfolio item

Portfolio Filters

<ul>
    <li class="active" data-filter="*">All</li>
    <li data-filter=".brand">Brand</li>
    <li data-filter=".design">Design</li>
    <li data-filter=".graphic">Graphic</li>
</ul>

Portfolio filters are the list item when clicked it will show portfolios related to those category.
data-filter : attribute specify the category the portfolios belongs to. It can have any value but thing to notice is that the value starts with ".value".

Portfolio Item

<figure>
    <img src="img/portfolio/img-*.jpg" alt="">
    <figcaption>
        <h3><!-- Project Name --></h3>
        <h4>View Detalis</h4>
        <p>
            <i class="fas fa-filter"></i><!-- Category -->
        </p>
        <a class="*-link" href="img/portfolio/img-*.jpg"></a>
    </figcaption>
</figure>

Portfolio item is contain inside <div> tag. It have class attribute which value are item and brand, item is used for styling the portfolio single item and brand is used to show which category the item belongs to in respect to the filter.

Portfolio item has inside anchor tag. This tag has class and href attribute. The class attribute specify that the link is of image type. There are 3 possible values:

  • image-link : This tells the scripts to show as image type and will show as popup when click.
  • video-link : This tells the scripts to show as video type and will show as popup when click.
  • ajax-link : This tells the script to show as ajax type and slide in as a page. Ajax links are pages like single-work.html which the script will import when the item is clicked.

Contact Section

This section contains contact form, contact info, google map.

Contact Form: Form contains the form fields for the user to send message. Form is based on ajax when submit the form will run the mail.php file. If everything is valid the mail will be send to the person whose email is specified in mail.php file.

Note: Edit this line in mail.php

$to = "you-email@example.com"; <!-- Change the email to your email id -->

Google Map: Enter the latitude and longitude of your location.

<div id="map" class="map" data-latlog="[<!-- Your Latitude-- >, <!-- Your Longitude -->]" data-popuptext="<!-- Address -->"  data-popupicon="img/marker.png"></div>


Template Elements

YouTube Video

In order to insert your video from youtube, change the link to it:

<div id="video-container">
    <div id="play-video" class="player" data-property="{videoURL: '<!-- Your Link -->' ...></div>	
</div>

Html5 Video

To insert your video, upload your video to the 'video' folder:

<div class="html-video-container" data-vide-bg="video/<!-- The name of your video -->.mp4"></div>


CSS Structure

  • css
    • plugins
    • bootstrap.css
    • font-awesome.css
    • magnific-popup.css
    • simplebar.css
    • swiper.min.css
    • jquery.animatedheadline.css
    • map.css
    • setting
    • *-color.css
    • circle.css
    • hexagon.css
    • *-title.css
    • style-dark.css
    • style-light.css

style-dark.css: This file is the main CSS file for this template and contains a Dark style.
style-light.css: This file is the main CSS file for this template and contains a Light style.

(*-color.css): This files are for choosing highlighed color of you own choice. There are 6 color file in this template.

(*-title.css): These files are for selecting the header display box.

(circle.css) and (hexagon.css): These files are for choosing a box style. This template has 2 box styles.

All other css files are designed to maintain the basic functions of this template.

Note: Editing these files can cause unexpected behaviour to the template only edit these file if you know what you are doing.
